| Orden | Archivo         | Qué haces ahí                                         | Motivo                                        |
| ----- | --------------- | ----------------------------------------------------- | --------------------------------------------- |
| 1️⃣   | `app.html`      | Empiezas a construir la estructura visual (tu página) | Es lo que se muestra en pantalla              |
| 2️⃣   | `app.scss`      | Das estilo a esa vista                                | Controla solo los estilos del componente raíz |
| 3️⃣   | `app.ts`        | Añades lógica y funcionalidad                         | Controla la interacción de la página          |
| 4️⃣   | `styles.scss`   | Defines estilos globales                              | Afecta toda la app                            |
| 5️⃣   | `app.routes.ts` | (Más adelante) Añades navegación entre páginas        | Si tu Padlet crece en secciones               |
