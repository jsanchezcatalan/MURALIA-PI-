⚙️ ARCHIVOS DE CONFIGURACIÓN ANGULAR
Estos archivos controlan la compilación, dependencias y configuración general del proyecto.
No deben modificarse sin conocer su función y no permiten comentarios dentro (formato JSON estricto).

- angular.json → Configura el build, serve, test y assets.
- package.json → Lista dependencias y scripts.
- package-lock.json → Fija versiones exactas de dependencias.
- tsconfig*.json → Define opciones del compilador TypeScript.
